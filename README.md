# PROFino
A simple profiler for Arduino.

# Dependencies
* avr-libc ```apt install avr-libc```
* avrdude ```apt install avrdude```
* ctags-universal ```apt install universal-ctags```
* serial (python) - ```pip install serial```

# Usage 

```./PROFino.py source.c```

or

```python PROFino.py source.c```
